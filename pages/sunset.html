DON'T MISS THE SUNSET TODAY

Everybody loves a beautiful sunset, it would be a shame to miss the chance to see it. With this website not only can you check how much time is left until the sunset in your city, but also in any place in the world.

SEARCH FOR ANY CITY

The functionality of the application is very straightforward. When you search for a city, the sunset time and geographical coordinates of the queried location are fetched from IP Geolocation Astronomy API. Next the local time is fetched from Timezone DB API to compare the two and display a countdown. This way the output is correct no matter the time zone.

GET READY

A detail that I particularly enjoy is that every time you search for a city, a new picture of a sunset is displayed in the background, which gets the user excited to see one with their own eyes. It works thanks to the use of the Unsplash API which fetches a random picture from their database with the keyword 'sunset'.
