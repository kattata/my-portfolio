PARKSHARE

A thesis project, where I collaborated with my internship company to develop a prototype for their product using web technologies.

FIND A PARKING SPOT THAT WORKS FOR YOU

ParkShare helps drivers find parking spots to rent posted by private owners which allow them to waste less time in traffic, reduce CO2 emissions and save money. My job was to prepare a proposition of a mobile app that could automate their tasks.

SEARCH FOR YOUR ADDRESS AND BROWSE THE MAP

The first thing a user will interact with is the map. Thanks to MapBox API and ReactMapGL, I marked all the available parking spots with tags, which immediately show the user the price.

The search function utilises PositionStack API, which suggests a list of real locations based on the typed in query. This helps the user be sure that the address they choose exists.

SELECT BOOKING INFO

When the address is chosen, the user may select the booking dates in the slide up modal. These can range anywhere from 20 minutes to multiple weeks or even months, depending on the user's needs.

Next, the user has the option to choose, which vehicle he wants to park to make sure the spot owner can identify the correct car. The user can select from his existing vehicles or add a new one to match his circumstances.

PARKING SPOT DETAILS

When all booking details are filled out, the user explores available spots and taps on his favorite one to see more information about it, such as spot size, description or any extra commodities. They can also confirm their booking time, which was saved in a state during the previous step.

When a parking spot is chosen, the user may press "Book now" and receive a confirmation.
